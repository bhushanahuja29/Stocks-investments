## 🎭 Animation Showcase

### 1. Page Load Sequence
```
Time    Animation
────────────────────────────────
0ms     Start
100ms   Market selector fades in ↓
200ms   Symbol pills slide up ↑
300ms   Price hero slides up ↑
400ms   Level card 1 fades in ↓
450ms   Level card 2 fades in ↓
500ms   Level card 3 fades in ↓
550ms   Level card 4 fades in ↓
600ms   All animations complete ✓
```

### 2. Price Update Animation
```
Before:  $45,234.56
         ↓
Flash:   $45,234.56  (scale 1.02, opacity 0.6)
         ↓
After:   $45,234.56  (scale 1.0, opacity 1.0)

Duration: 300ms
```

### 3. Alert Toggle Animation
```
OFF State:  [○─────────]  🔕
            ↓
Animating:  [─○────────]  (thumb sliding)
            ↓
ON State:   [─────────●]  🔔

Duration: 300ms
Background: Gray → Green
```

### 4. Progress Bar Animation
```
Initial:    [░░░░░░░░░░] 0%
            ↓
Animating:  [████░░░░░░] (smooth width transition)
            ↓
Final:      [████████░░] 80%

Duration: 600ms
Effect: Shimmer overlay
```

### 5. Hover Effect
```
Normal:     Card at y=0, shadow-md
            ↓
Hover:      Card at y=-2px, shadow-lg
            Price color: primary → blue
            Stats bg: card → hover

Duration: 200ms
```

## 📊 Visual Hierarchy

### Information Priority
```
Level 1 (Highest):
  ├─ Current Price (64px, bold, center)
  └─ Symbol Name (24px, bold)

Level 2 (High):
  ├─ Level Prices (32px, bold)
  └─ Distance Indicators (colored badges)

Level 3 (Medium):
  ├─ Stats (15px, grid layout)
  └─ Progress Bars (visual indicator)

Level 4 (Low):
  ├─ Timestamps (14px, gray)
  └─ Labels (12px, uppercase)
```
